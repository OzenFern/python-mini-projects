# English to Pig Latin

Implementation of a simple English to Pig Latin converter.

## Rules

1. If word begins with a consonant, the consonant is moved at the end of the word and -ay is added as a suffix.
2. If words begins with a vowel, then -yay is added at the end of the word.
3. Numbers are left untouched.
4. Case of the first letter is kept the same.

### Examples

- cat -> atcay
- eat -> eatyay
- 542 -> 542
- I love Python! -> Iyay ovelay Ythonpay!
