# Blind Auction (Python)

A simple terminal-based **blind auction program** written in Python.  
It collects bids from multiple participants and determines the highest bidder.

---

## About

This program simulates a blind auction where multiple users can enter their names and bid amounts.  
Each bidder's information is stored in a dictionary, and once bidding ends, the program determines the winner based on the highest bid.

The screen clears between entries so bidders cannot see previous bids.

---

## How It Works

- A bidder enters their **name** and **bid amount**
- The program stores bids using a Python dictionary
- The screen clears before the next bidder enters
- Once bidding ends, the program calculates the **highest bid**
- The winner and bid amount are displayed

---

## Features

- Input validation for bid amounts
- Prevents duplicate bidder names
- Cross-platform screen clearing (Windows / Linux / macOS)
- ASCII logo displayed when the program starts

---

## Requirements

- Python  
- No external libraries required
